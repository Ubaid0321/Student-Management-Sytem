================================================================================
STEP 1: FIND YOUR COMPUTER'S IP ADDRESS
================================================================================
1. Open Command Prompt (Press Windows + R, type "cmd", press Enter)
2. Type: ipconfig
3. Find "Wireless LAN adapter Wi-Fi" section
4. Copy the "IPv4 Address" (e.g., 192.168.1.5)

================================================================================
STEP 2: UPDATE CONFIG FILE
================================================================================
1. Open file: mobile\config.js
2. Find this line:
   const YOUR_COMPUTER_IP = '192.168.100.19';
3. Replace with YOUR IP from Step 1:
   const YOUR_COMPUTER_IP = '192.168.1.5';  <-- Your IP here
4. Save the file

================================================================================
STEP 3: INSTALL BACKEND DEPENDENCIES
================================================================================
Open Command Prompt:

cd "C:\path\to\Student Managment system\backend"
npm install

Wait for installation to complete (1-2 minutes)

STEP 4: INSTALL FRONTEND DEPENDENCIES
================================================================================
Open another Command Prompt:

cd "C:\path\to\Student Managment system\mobile"
npm install

Wait for installation to complete (2-3 minutes)

If you see errors, try:
npm install --legacy-peer-deps

================================================================================
TROUBLESHOOTING
================================================================================

PROBLEM: "npm not found"
SOLUTION: Reinstall Node.js from https://nodejs.org and restart Command Prompt

PROBLEM: "expo not found"
SOLUTION: Run: npm install -g expo-cli

PROBLEM: Can't connect from phone
SOLUTION: 
  - Check phone and computer are on same WiFi
  - Disable Windows Firewall temporarily
  - Check IP address is correct in config.js

PROBLEM: Port blocked
SOLUTION: Allow Node.js through Windows Firewall
  1. Open "Windows Security"
  2. Go to "Firewall & network protection"
  3. Click "Allow an app through firewall"
  4. Find "Node.js" and check both Private and Public

PROBLEM: npx command blocked
SOLUTION: Run Command Prompt as Administrator

PROBLEM: Deprecated package warnings
SOLUTION: These are just warnings, the app will still work
